Welcome to the CRUD App

This is a simple CRUD application built with Go and SQLite.

CRUD stands for Create, Read, Update, and Delete. It represents the basic operations you can perform on a database or application.

We have containerized the application using Docker and docker-compose. For installation on Kubernetes, we use Helm to manage the application's lifecycle.

We have implemented endpoints for probes in the Go application and created an exporter to collect metrics from our application.
